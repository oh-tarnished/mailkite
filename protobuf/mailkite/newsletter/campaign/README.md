| Value | Number | Description |

| `CAMPAIGN_STATE_UNSPECIFIED` | 0 | Unset. |
| `CAMPAIGN_STATE_DRAFT` | 1 | Editable, not queued. |

### CampaignTransition

A lifecycle transition requested via TransitionCampaign.

| Value | Number | Description |
| --- | --- | --- |
| `CAMPAIGN_TRANSITION_UNSPECIFIED` | 0 | Unset. |
| `CAMPAIGN_TRANSITION_SCHEDULE` | 1 | Schedule a draft to start at its schedule_time. |
| `CAMPAIGN_TRANSITION_START` | 2 | Begin sending immediately. |
| `CAMPAIGN_TRANSITION_PAUSE` | 3 | Pause a running send. |
| `CAMPAIGN_TRANSITION_RESUME` | 4 | Resume a paused send. |
| `CAMPAIGN_TRANSITION_CANCEL` | 5 | Cancel the campaign permanently. |

### CampaignView

Controls how much of a campaign is returned by read methods.

| Value | Number | Description |
| --- | --- | --- |
| `CAMPAIGN_VIEW_UNSPECIFIED` | 0 | Default; equivalent to CAMPAIGN_VIEW_FULL. |
| `CAMPAIGN_VIEW_BASIC` | 1 | Omits the large body and alt_body fields. |
| `CAMPAIGN_VIEW_FULL` | 2 | Returns the full campaign, including body. |
