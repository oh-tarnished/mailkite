## Enums

### TemplateType

What a template can be applied to.

| Value | Number | Description |
| --- | --- | --- |
| `TEMPLATE_TYPE_UNSPECIFIED` | 0 | Unset. |
| `TEMPLATE_TYPE_CAMPAIGN` | 1 | Wraps regular campaigns. |
| `TEMPLATE_TYPE_CAMPAIGN_VISUAL` | 2 | Wraps campaigns authored in the visual editor. |
| `TEMPLATE_TYPE_TRANSACTIONAL` | 3 | Wraps transactional messages. |

### TemplateView

Controls how much of a template is returned by read methods.

| Value | Number | Description |
| --- | --- | --- |
| `TEMPLATE_VIEW_UNSPECIFIED` | 0 | Default; equivalent to TEMPLATE_VIEW_FULL. |
| `TEMPLATE_VIEW_BASIC` | 1 | Omits the body field. |
| `TEMPLATE_VIEW_FULL` | 2 | Returns the full template, including body. |
